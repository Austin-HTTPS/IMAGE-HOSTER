Coder100 Image Hoster

A simple way to host your screenshots.

About

Many people have image vaults, so I made this public "vault".

Usage

The hoster can be used for a variety of purposes, but it can be mainly used as a permanent way to store an image.

Hosting

Feel free to fork this project and rename it to a shorter link. Be sure to include the credits footer. Source Code.

Privacy

Your images are private unless someone has the link. They can also be found in the Source Code. The hoster is anonymous and we do not track uploader data.

File Names

File names are randomly generated. They contain a mix of alphanumerical letters including '-'.

Restrictions

Please do not post inappropriate images or copyrighted images. I will not be responsible for any uploaded images.

Storage Time

Images in theory will be stored forever, however the database may be cleared from time to time for maintainence purposes. Fork this project to have complete control over the image database and keep images forever.
